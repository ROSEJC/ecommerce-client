# TechZone クライアント

**TechZone** は、ヘッドホン販売のための EC サイトです。このリポジトリは、React と Tailwind CSS を使って構築されたフロントエンド部分です。Axios を通じてバックエンドと通信し、基本的な eコマース機能を実現しています。

## 主な機能

- 商品一覧と詳細の表示
- カートに追加して購入
- ユーザー登録とログイン
- 注文履歴とレビュー表示・投稿
- ダークモード対応

## 使用技術

- React
- Tailwind CSS
- Axios
- React Router DOM
- Context API
- Lucide-react（アイコン）

## セットアップ手順

```bash
cd client
npm install
npm run dev
```

フロントエンドはデフォルトで `http://localhost:3000` のバックエンドに接続します（`.env` ファイルで変更可能）。
